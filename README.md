# Ember CLI Full Calendar

This is a *WIP* Ember wrapper for jQuery FullCalendar plugin.

## Usage

Use the `full-calendar` component -

```{{full-calendar events=events}}```

### Supported Options

* headerLeft
* headerCenter
* headerRight
* theme
* firstDay
* isRTL
* weekends
* hiddenDays
* fixedWeekCount
* weekNumbers
* height
* editable
